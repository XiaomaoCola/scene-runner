# data/

此目录存放所有外部数据文件，**不入 git repo**（通过 外置储存 同步）。

克隆仓库后，把外置储存里的对应文件夹拖回此处即可。

## templates

这个文件夹里主要放 `opencv` 对照用的模板。

## scratch

草稿区，随时可删，根目录的 `tools` 文件夹里的工具产出的内容一般进这个文件夹。

## resource

在`30_templates`的文件夹里，有builder base的右上角资源的数字模板，用来识别builder base里右上角金币和圣水数量。

## models

这个文件夹里放训练好的模型。
