# 文件导航

## configs/intents/

各 Plan 的 UI 区域坐标配置，使用 0–1 归一化坐标，与分辨率无关。
（但是也尽量把游戏运行在1920*1080分辨率下，因为别的分辨率下我还没测试过。）

### BuilderBaseAttack/

| 文件 | 说明 |
|------|------|
| `stage1_builder_base.yaml` | 村庄主界面：攻击按钮、资源区域等 |
| `stage2_attack_menu.yaml` | 攻击菜单：Find Now! 按钮 |
| `stage3_battle_scene.yaml` | 战斗界面：部署区域、兵种栏、投降按钮等 |
| `stage4_surrender_confirm.yaml` | 投降确认弹窗：OK 按钮 |
| `stage5_return_home.yaml` | 战斗结算界面：Return Home 按钮 |

### BuilderBaseCollectResources/

| 文件 | 说明 |
|------|------|
| `stage3_collect_and_return_home.yaml` | 收集弹窗：Collect 按钮、关闭按钮 |
